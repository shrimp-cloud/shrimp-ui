# shrimp-ui
前端UI组件

### 安装
```shell
# 安装依赖
pnpm install

# 构建
pnpm build
```

### 发布
1. 升级version 版本号
2. 推送参考文档：https://packages.aliyun.com/npm/npm-registry/guide
3. 推送：
```
pnpm publish
```


### 依赖
```shell
# 安装依赖
yarn add shrimp-ui
```

### 使用示例
```javascript
import Pagination from 'shrimp-ui/es/components/Pagination'

```
